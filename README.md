# quant-research
项目简介：
       从单因子策略以及改进后采用多因子策略在基于沪深300下的回测，。
项目| 技术 | 回测区间 | 策略收益 | 基准收益 |：
 低PE单因子策略(./strategy/low_pe_strategy.py) | Python/聚宽 | 2019-2026 | -39.72% | +49.55% |
 多因子选股策略(./strategy/multi_factor_strategy.py) | Python/聚宽/排名法 | 2019-2026 | +53.47%| +49.55% |



 低PE单因子策略归因分析：因子构成：低PE，选股方式：每月调仓，等权持有10只股票，基准：沪深300指数

        市场风格：2019-2026年A股市场整体偏向成长风格，低PE价值因子持续跑输
                低PE vs 高PE的表现对比：
                                        时间	持有的低PE股票表现	  同期高PE成长股（如新能源、AI、芯片）
                                        2019	钢铁、地产反弹有限	  消费、科技大涨
                                        2020	银行、地产滞涨	      医药、新能源爆发
                                        2021	地产暴雷（华夏幸福、蓝光等）	新能源、周期股（煤炭除外）上涨
                                        2022-2023	地产链持续下跌	  AI、数字经济大涨
                                        2024-2026	银行、地产筑底但未反转	科技股分化，但整体仍优于价值

        行业：策略重仓银行、地产、钢铁、煤炭等顺周期板块

             房地产集中度过高：2019-2021年期间，房地产行业在持仓中占比长期在30%-50%以上。2021年后房地产行业整体暴雷，严重拖累了策略收益。

             钢铁、煤炭等周期股：虽然部分时期（如2021年）有过反弹，但长期来看，这些行业的低PE往往是“价值陷阱”——盈利能力下降导致PE被动升高，但股价持续下跌。

             银行股的低PE陷阱：银行股PE长期在4-6倍，但股价长期不涨，甚至下跌。低PE未能转化为股价上涨。

             
多因子选股策略：因子构成：低PE(40%) + 高ROE(40%) + 20日动量(20%)，选股方式：每月调仓，等权持有10只股票，基准：沪深300指数
       
       总体表现：微弱跑赢基准，超额收益主要来自少数几只股票（神火股份、云铝股份、赛轮轮胎）
                超额收益来源：
                            质量因子（高ROE）成功规避了价值陷阱，低PE单因子重仓银行、地产（PE低但ROE也在下降）多因子策略要求高ROE，所以排除了这些表现不佳的低PE股
                            股票	低PE单因子持仓	多因子策略持仓	原因
                            华夏幸福	✓（PE 4倍）	✗（ROE下降）	2021年暴雷，跌80%
                            蓝光发展	✓（PE 5倍）	✗（ROE为负）	2021年暴雷，跌90%
                            神火股份	✗（PE 8-10倍不算最低）	✓（ROE 20%+）	2021-2022年涨200%
                            云铝股份	✗（PE中等）	✓（ROE 15%+）	2023-2024年涨150%

                            动量因子抓住了结构性行情：贡献表现：
                            2021年：动量因子捕捉到煤炭化工（神火、氯碱）的上涨趋势
                            2023-2024年：动量因子捕捉到铝业（云铝）、轮胎（赛轮）的景气周期
                            2024年底：动量因子捕捉到农业股（京基智农）的上涨

最终结论：多因子策略相对于低PE单因子的改进是成功的：收益从-39.72%提升到+53.47%，成功规避了地产链暴雷，抓住了周期股中的景气机会。
